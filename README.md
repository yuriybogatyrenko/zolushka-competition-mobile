Gulp compiler
======

Description
-----------
This is some project that is done with some technologies like gulp, twig, sass, etc.

Deployment
----------
1) clone repository:
```
git clone the repository
```
2) choose release if necessary;

3) install backend dependencies:
```
npm install
```
4) install frontend dependencies:
```
bower install
```
5) launch watchers
```
gulp
```
